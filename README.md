# Yang Xianghao Portfolio

这是杨翔皓的个人展示页，主题是测试开发、云平台质量保障、自动化回归、Kubernetes/Linux 排障、AI 辅助测试和测试左移。

## 文件结构

- `index.html`：正式静态页面入口。
- `assets/`：简历 PDF 等公开资源。
- `PRD.md`：页面需求与设计说明。
- `output/`：桌面端、移动端和重点区块截图。

## 本地预览

直接用浏览器打开 `index.html` 即可预览。页面为纯静态实现，不依赖构建工具。

## 部署

可以直接部署到 GitHub Pages、Sealos 静态站点或任意静态资源托管服务。部署前建议确认 `assets/` 中的 PDF 是否适合公开分发。

## 隐私说明

页面不直接展示头像，也不暴露私有仓库链接。涉及私有仓库的项目仅保留脱敏描述，并标注“可提供脱敏说明”或“可提供脱敏演示”。当前页面结尾只展示姓名、城市、手机号和邮箱，手机号与邮箱点击后复制到剪贴板，不跳转到电话或邮箱客户端。
